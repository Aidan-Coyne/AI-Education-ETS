# AI Strategy & Engineering Lecture Companion Website

This repository contains a high-impact, visual, scrolling landing page designed as a lecture handout/companion guide for workshops on AI Solutions Engineering and Product Delivery.


## Live Website & Hosting

This site is optimized to be hosted **completely for free** using **GitHub Pages**. 


## File Structure

- `index.html`: The HTML skeleton containing the 7 core lecture sections, loaded with SEO tags and custom Google Fonts.
- `style.css`: Modern styling variables, custom vertical timelines, grid columns, license comparison blocks, and smooth animations.
- `script.js`: Vanilla JavaScript handling:
  - **ScrollSpy:** Automatically highlights the active section in the sticky nav header as you scroll.
  - **IntersectionObserver:** Triggers slide-up, fade-in, and connection line animations as sections enter the viewport.
  - **Lightbox Viewer:** Opens high-resolution previews of timeline images in a full-screen overlay when clicked.
- `*.jpeg`: Local portfolio, lecture, and engineering JPEGs displayed dynamically in Section 1.
